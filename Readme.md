markdown# 🚀 Logistic Regression

Predicting breast cancer type using a Logistic Regression model.

## 🛠️ Tech Stack
- **Language:** Python
- **Machine Learning:** Scikit-learn (sklearn)
- **Data Manipulation:** Pandas

## 🏁 Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites
Ensure you have Python installed on your system, along with the following libraries:
- pandas
- scikit-learn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/amuzalda/Machine-Learning.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Machine-Learning
   ```
3. Install the required dependencies:
   ```bash
   pip install pandas scikit-learn
   ```

## 💻 Usage
Run the main script to train the model and see predictions:
```bash
python LogisticRegression.py
```
